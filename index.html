리모델링 견적 계산기

리모델링 견적 계산기 평당 + 품목 + 옵션

상위: 평수×단가(기본/표준/고급) / 하위: 도배·장판(평당) + 싱크대·욕실·필름(정액) + 옵션(가산/할인)

① 기본 공사비 (평당)

② 품목 공사비 (추가)

도배 (선택 단가/평)
평수 기준 자동 계산
벽상태가 고르지 못할 때 추가비용 발생 (현장답사 후 결정)
바닥 (선택 단가/평 + 밑작업 옵션)
평수 기준 자동 계산
장판/데코타일: +10,000원/평 · 마루: +15,000원/평 · 타일/대리석: 기본 0원(필요시 현장 결정)
붙박이장 (자당 180,000원)
‘자’ 숫자만 입력 (예: 8)
싱크대(기본형) (M당 900,000원)
수전/싱크볼/악세사리 별도 · 미터 숫자만 입력
신발장 (M당 900,000원)
미터 숫자만 입력 (예: 2.4)
욕실 (선택 금액/개소)
등급 선택 + 개소수 입력
필름 (1,200,000원)
정액 세트 + 종류 선택

③ 옵션 (가산/할인)

철거 (평당 120,000원)
평수 기준 자동 계산
폐기물 처리 (+400,000원)
폐기물/마대
현장 난이도 (+5%)
엘베 없음 등
프로모션 할인 (-300,000원)
기간 할인
평수와 항목을 선택한 뒤 “견적 계산”을 눌러보세요.
